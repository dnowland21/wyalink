# WyaLink Logos

Place your WyaLink logo files here:

- `wyalink-logo.svg` - Main color logo
- `wyalink-logo-white.svg` - White version for dark backgrounds
- `wyalink-icon.svg` - Icon/symbol only
- `wyalink-wordmark.svg` - Text-only version

These will be accessible in your app via `/logos/filename.svg`
